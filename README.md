# Sistema de Ventas - Java + MySQL
Proyecto personal desarrollado en Java y MySQL con arquitectura en capas (DAO / Service / Model) orientado a la gestión de usuarios y productos para un sistema de ventas.

El objetivo principal del proyecto es reforzar conocimientos de desarrollo backend utilizando Java vanilla, JDBC y buenas prácticas de programación orientada a objetos.

Actualmente el sistema cuenta con:

- Autenticación de usuarios
- Manejo de roles (ADMIN y VENDEDOR).
- Seguridad de contraseñas con BCrypt.
- CRUD de usuarios.
- Gestión de productos.
- Activación/desactivación lógica.
- Validaciones de negocio en capa Service.
- Arquitectura por capas.
